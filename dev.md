## 环境说明

- 系统：windows 10 或 windows 11
- nodejs：v20

## 安装

```bash
pnpm install
```

## 开发

1. 运行扩展，在 vscode 下按`F5`
2. 插件运行时，查看 webview 调试日志：`Ctrl + Shift + I`

## 打包

```bash
pnpm package
```
